❓ Help - Web Activity Tracker

📋 What This Extension Does

This Chrome extension tracks your web browsing activity and shows usage statistics for the last:

  • 1 Day
  • 7 Days
  • 30 Days

It helps you understand where your time is going online.

🔍 Features

  • Track time spent on each domain
  • Daily, weekly, and monthly breakdown
  • View most-visited sites
  • Lightweight and privacy-respecting (data stored locally)

⚙️ How to Use

  1. Install the extension from the Chrome Web Store.
  2. Click the extension icon to open the dashboard.
  3. Select a time range: 1Day / 7Days / 30Days.
  4. Review your activity summary.

🔐 Privacy Notice

Your data is stored locally in your browser. It is never sent to external servers.
